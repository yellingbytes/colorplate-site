## App Review Notes

- No account is required.
- If demo data is enabled for review, mention the route or launch flag in App Review notes.
- Explain that ColorPlate is a visual food archive, not a calorie tracker, diet app, medical device, diagnosis tool, or nutrition planning product.
- Explain how photo analysis works, including whether a build uses local analysis, third-party AI processing, or a user-provided API key.
- If location-based food discovery is enabled, state that location is optional and only used when the user requests nearby suggestions.

## Privacy Details To Keep In Sync

- Meal photos selected by the user.
- Generated meal analysis, color distributions, scores, streaks, and archive artwork.
- Optional location for nearby discovery.
- Optional support emails and privacy requests.
- Any analytics or crash reporting if added later.
- Any third-party AI provider used for image analysis.

## Compliance Notes

- Regulatory medical device status: ColorPlate should be declared as not a regulated medical device unless its feature set changes materially.
- Export compliance: answer according to the encryption and network behavior of the submitted build.
- Age rating: answer based on actual app content and whether unrestricted web/location features are enabled.
- Accessibility Nutrition Labels: complete based on the submitted app experience.
